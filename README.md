# Project Member (Magang)

Proyek ini dikembangkan menggunakan **CodeIgniter 3 (CI3)** sebagai framework utamanya.  
Tujuannya adalah untuk memberikan **informasi kepada pelanggan mengenai layanan yang tersedia** di sistem.
